Markdown Syntax Guide

Headings

Headings are used to mark the importance of a title.

# Headings Level 1 ## Headings Level 2 ### Headings Level 3 #### Headings Level 4 ##### Headings Level 5

Paragraphs

In Markdown doesn't need to mark a tag for create paragraphs.

This is a paragraph

For line breakings just need to declare a \ Let's create a line break\
Now it's created

Emphasizes

It's possible to mark a word or a phrase like bold or italic in Markdown.


Bold: I'm a **phrase** Italic: Do you like a *italic* word? I'm am too a _italic_ word Bold and Italic: I'm ***Batman***

Blockquotes

It's easy and simple to declare a Blockquote, just use a >.

> I have a dream, one day... Paragraph Blockquote: > 1
>
> 3 Nested Blockquotes: > 1
>> 2.1
